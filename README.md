# jarvil

**j**ust **a** **r**eal **v**iable **i**ntuitive **l**auncher

## installation

First of all, you need electron installed globally:

```bash
npm i -g electron
```

After this you need to install the project dependencies:

```bash
npm i
```

or

```bash
yarn
```

## development

```bash
npm start
```

or

```bash
yarn start
```

## build

for your current used system

```bash
npm build
```

or

```bash
yarn build
```
